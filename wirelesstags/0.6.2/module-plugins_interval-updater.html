Module: plugins/interval-updater

This is a TagUpdater plugin implementation that simply uses the WirelessTag#startUpdateLoop and WirelessTag#stopUpdateLoop methods, which auto-update a tag based on the update interval configured for it.

The main motivation is to have a module for using an interval-based update method that uses the same API interface as others, and can thus be swapped in.
